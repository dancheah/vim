This is my vim configuration.

__Installation:__  

* git clone git@github.com:dancheah/vim.git ~/.vim
* cd ~/.vim && git submodule init
* cd ~/.vim && git submodule update
* mkdir ~/.vim/tmp
* cd ~/ && ln -s .vim/vimrc .vimrc 
* cd ~/ && ln -s .vim/gvimrc .gvimrc
